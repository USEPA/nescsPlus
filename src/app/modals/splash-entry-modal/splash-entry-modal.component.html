The National Ecosystem Services Classification System (NESCS) Plus is a system for classifying Final Ecosystem Services (FES). FES are specifically defined as the services from nature that are “directly [emphasis added] enjoyed, consumed, or used to yield human well-being” (Boyd and Banzhaf, 2007).

  • NESCS Plus offers two options for searching.
  1. "Browse Core Options" refines the >75,000 possible combinations to a shorter "core" set of ~1000 most plausible combinations. If you discover a row that is missing that should be included in the core "core" set, please let us know by clicking "Contact Us."
  2. In contrast, "Query All Options" enables users to consider every possibility and build a custom table a single row at a time.
