NESCS Plus offers two options for searching. (1) These are the results from "Browse Core Options" which refines the >75,000 possible combinations to a shorter "core" set of ~1,000 most plausible combinations. (2) In contrast, the “Query All Options” enables users to consider every possibility and build a custom table a single row at a time. If you discover a row that is missing that should be included in the "core" set, please let us know by clicking "Contact Us."
